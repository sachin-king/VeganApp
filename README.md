# 🌱 Vegan Finder App

An Android application that helps users discover delicious vegan recipes and vegan-friendly restaurants.
The app is designed with a clean UI and provides categorized recipes for easy browsing.

## 📱 Features

* 🍽 Browse different vegan recipes
* 🏪 View vegan restaurants
* 🔎 Search functionality to find recipes easily
* 📂 Categories for better navigation
* ⭐ Modern and simple user interface

## 🛠 Tech Stack

* **Language:** Kotlin
* **IDE:** Android Studio
* **UI:** XML Layouts
* **RecyclerView** for displaying lists
* **Firebase** (if used)
* **Google Maps API** (for restaurant locations)

## 🚀 Installation

1. Clone the repository

```
git clone https://github.com/yourusername/VeganApp.git
```

2. Open the project in Android Studio

3. Sync Gradle and run the app on an emulator or Android device.
